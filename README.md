# Repositorio TP2 - Programación Móvil  

Repositorio que contiene los mini proyectos realizados a lo largo del trabajo práctico 2 de la materia "Programación Móvil" del año 2026. El mismo está organizado con la siguiente estructura:
- Ejercicio
- - Subejercicio
- - - Autor
    
Esto con el objetivo de que cada persona pudiese hacer los ejercicios que quisiese, adicionales a los obligatorios. Entonces, por ejemplo, un ejercicio que haya sido realizado por dos personas, tendrá las siguientes rutas:
- Ejercicio2/EjercicioB/Persona1/ y Ejercicio2/EjercicioB/Persona2/ con el código del ejercicio.
